# Технический бэклог «Schoolify»

Задачи разбиты по модулям и по сложности: 🟢 лёгкая · 🟡 средняя · 🔴 сложная.
Каждая задача привязана к feature-ветке из ТЗ.

## Легенда сложности
- 🟢 **Лёгкая** — конфиг/верстка/CRUD-заглушка, до полудня работы.
- 🟡 **Средняя** — логика с несколькими слоями, валидация, интеграция фронт-бэк.
- 🔴 **Сложная** — бизнес-логика с транзакциями, безопасность, нетривиальные связи.
